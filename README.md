# Python Data Processing and Analysis Assignment

## Overview

This repository contains solutions for 6 real-world data processing problems using Python.
The assignment demonstrates usage of core Python concepts such as:

* Lists
* Dictionaries
* Strings
* Tuples
* Loops and Conditions
* Basic Data Analysis Logic

All programs are written and executed in a Jupyter Notebook and outputs are included.

---

## Problem Statements

### Problem 1: Employee Performance Bonus Eligibility

Find employees with the highest performance score and handle tie cases.

### Problem 2: Search Query Keyword Analysis

Analyze search queries, count keyword frequency and display repeated words.

### Problem 3: Sensor Data Validation

Filter valid sensor readings (even numbers) and store hour-value pairs.

### Problem 4: Email Domain Usage Analysis

Count number of users per email domain and calculate percentage usage.

### Problem 5: Sales Spike Detection

Detect days where sales increased approximately 30% above average.

### Problem 6: Duplicate User ID Detection

Identify duplicate user IDs and count their occurrences.
